# Project Images

This folder contains images and photos related to the Road Inventory Straightline Diagram project.

## Usage

Images stored here will be accessible via GitHub's raw file hosting:

```
https://github.com/kyrhyl/SLD/raw/master/images/filename.jpg
```

## Recommended Image Types

- **Screenshots** of the application
- **Diagrams** and flowcharts
- **Icons** and UI assets
- **Documentation images**

## File Size Guidelines

- Keep individual images under **10MB** for better performance
- Use appropriate compression for photos
- Consider using WebP format for better compression

## Adding Images

1. Place your image files in this `images/` folder
2. Commit and push to GitHub:

```bash
git add images/
git commit -m "Add project images"
git push
```

## For Large Photo Collections

If you have many large photos, consider using dedicated photo hosting services:

- **Imgur**: Unlimited uploads, fast access
- **Google Photos**: Unlimited storage for personal photos
- **Flickr**: Good for organized photo collections
